## Installation

- Downloads
```
▖~ git clone https://github.com/aniruddha0pandey/FindByFace.git && cd FindByFace
▖~ 
```
- Build
```
▖~ npm install
▖~ npm start
```
<sub>Navigate to http://localhost:3000/ if not automatically redirected.</sub>
- Deploy
```
▖~ npm install -g serve
▖~ serve -s build
```
